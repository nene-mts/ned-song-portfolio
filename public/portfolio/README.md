# Portfolio Assets

Put final, web-ready assets in these folders so the site can reference them with paths like `/portfolio/missgreen/cover/cover.jpg`.

Suggested use:
- `_shared`: reusable assets across projects.
- `personal/portrait`: headshots or profile photos.
- `personal/lifestyle`: casual photos for the Story section.
- `missgreen`: MissGreen project assets.
- `ebest`: eBest project assets.
- `tiktok`: TikTok internship or campaign assets.
- `woolworths`: Woolworths digital, UX, or case-study assets.

Inside each project:
- `cover`: project card cover image.
- `posters-visuals` / `brand-identity`: static visual work.
- `video-motion`: videos, reels, motion exports.
- `campaign-strategy`: campaign assets and strategic documents.
- `website-digital` / `website-app` / `product-screens`: web, product, or digital interface work.
- `research-process` / `analytics-results`: process, audit, or results screenshots.
